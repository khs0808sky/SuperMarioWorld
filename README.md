# SuperMarioWorld
> **2D Platform Game Portfolio**<br>

- **게임규칙** <br>
움직이는 몬스터(밤바, 땡감바)를 제한시간(100초)안에 모두 잡으면 클리어<br><br>
- **조작법** <br>
  
|키|조작|
|:---|:---|
|←|왼쪽 이동|
|→|오른쪽 이동|
|↑|위 보기|
|↓|쭈그리기|
|Z|스핀 점프|
|X|점프|
|C|달리기|
|R|게임오버 or 게임클리어 시 재시작|
